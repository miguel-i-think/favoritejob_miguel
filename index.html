The Purpose of this Page

My name is Miguel Gonzaga. I'm creating this page to share my personal favorite poject I've worked on as well as some details. To start of with, I've never worked in a company nor have I ever made a large scale project as a job. All of my coding projects have been purely for education. My favorite project is also very simple, but it holds a lot of personal value as I hope you'll see.

What Did it Entail

My favorite project was very simple. It was a python based version of the classic game 'Snake'

Although this might seem like a strange pick especially now after a couple years of programming, I had picked up Python as my first programming language and completed this project not long after learning the language. In other words, this was my first personal programming project.

Example of Project

The Challenges

  1. I had never programmed something with PyGame
  2. Even though I knew the basics of Python, they could only take me so far
  3. Wasted a lot of time looking up how to do things
  4. Had to balance making the game and doing school work
  5. My lack of use of comments made going back to fix things near impossible

A lot of time was spent re-reading the code over and over to figure out what did what. This led to some sections having to be rewritten (with comments added on the way), a lot of days where nothing was accomplished, rushed shcool work, and overall frustration. The important part in my eyes, is that despite all this, I pushed through it all.

The Reason

If there was so many problems and it was so frustrating, why would this be anyone's favorite project? Most people would pick a project they struggled with but always saw as a challenge. In my case this project truly felt like hell on Earth, especially when I realized I knew nothing. I was completely lost. In truth, this project was not fun a lot of the time and it felt like nothing but a task.

So why choose it? Clearly I don't like this project even now, right? Well in truth, this was my stepping stone. This is when I realized, that yes I don't know anything but that doesn't mean I have to stop. I saw it through to the end and actually managed to finish it. This gave me hope that I could one day achieve my goal of attaining a job in this field. After this I started studying harder, I started passing my programming classes with 90s - 100s, and the more I learned the easier everything else got. It was an amazing stepping stone that helped me prove to myself that it was possible. That's the reason I'm so proud of this project and am willing to call it my favorite even after all these years.
